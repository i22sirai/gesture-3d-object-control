# 手のジェスチャーによる3Dオブジェクト操作インターフェース

## 概要

Webカメラで取得した手の動きを用いて、Unity上の3Dオブジェクトを
操作するインターフェースを開発しています。

MediaPipe Handsで手のランドマークを取得し、Python側で処理した
情報をUDP通信によってUnityへ送信しています。

本研究では、キーボードやマウスを使わず、手の動きによって
3Dオブジェクトを直感的に操作できるインターフェースを目指しています。

## 使用技術

- Unity
- C#
- Python
- MediaPipe Hands
- OpenCV
- UDP通信
- Webカメラ

## システム構成

Webカメラ  
↓  
Python / MediaPipe Hands  
↓  
手のランドマーク取得・操作情報の生成  
↓ UDP通信  
Unity / C#  
↓  
3Dオブジェクトへの操作反映

## 現在の実装状況

現時点では、次の操作を実装しています。

- 両手の距離変化を利用した拡大縮小
- 手の動きを利用した3Dオブジェクトの回転
- PythonからUnityへのUDP通信
- Unity上の3Dオブジェクトへのリアルタイムな反映

現在は、拡大縮小と回転操作の安定性を改善しています。

## 今後の実装予定

- オブジェクトの選択
- オブジェクトの移動
- 誤操作を抑えるための判定処理
- 操作を段階的に整理する3層構造
- 操作方法の比較評価

3層構造を導入することで、限られたジェスチャーから複数の
3D編集操作を扱いやすくすることを目指しています。

## 研究上の着眼点

単にオブジェクトを動作させるだけでなく、ユーザーの操作意図を
どのように3D空間上の変化へ対応させるかを重視しています。

特に、初めて使用する人でも理解しやすく、誤操作の少ない
インターフェースの設計を目指しています。
