## 概要

Webカメラで取得した手の動きを用いて、Unity上の3Dオブジェクトを操作するインターフェースを開発しています。

MediaPipe Handsを用いて手のランドマークを取得し、Python側で生成した操作情報をUDP通信によってUnityへ送信します。Unity側では、受信した情報を3Dオブジェクトの拡大縮小と回転に反映します。

本研究では、キーボードやマウスを使わず、手の動きによって3Dオブジェクトを直感的に操作できるインターフェースの実現を目指しています。

本プロジェクトは、卒業研究として開発中のプロトタイプです。

---

## 研究の背景・目的

一般的な3D編集ソフトでは、キーボードやマウスを用いて、操作対象や機能を選択しながら編集を行います。そのため、初めて使用する人にとっては、操作方法を覚えるまでに時間がかかる場合があります。

本研究では、手のジェスチャーを入力として用いることで、初めて使用する人でも操作内容を理解しやすく、学習負担や誤操作の少ない3D操作インターフェースを目指しています。

特に、ユーザーの手の動きや操作意図を、どのように3D空間上の変化へ対応させるかを重視しています。

## システム構成

```text
Webカメラ
    ↓
Python / OpenCV / MediaPipe Hands
    ↓
手のランドマーク取得
    ↓
ジェスチャーおよび操作情報の生成
    ↓ UDP通信
Unity / C#
    ↓
3Dオブジェクトへの操作反映
```

Python側では、Webカメラの映像からMediaPipe Handsを用いて手のランドマークを取得します。

取得した両手の位置関係や距離の変化から操作情報を生成し、UDP通信によってUnityへ送信します。

Unity側では、Pythonから受信した情報をもとに、3Dオブジェクトの拡大縮小や回転をリアルタイムに反映します。

## 実行方法

### 1. リポジトリを取得する

Gitを使用する場合は、次のコマンドを実行します。

```bash
git clone https://github.com/i22sirai/gesture-3d-object-control.git
```

取得したフォルダへ移動します。

```bash
cd gesture-3d-object-control
```

Gitを使用しない場合は、GitHub上の「Code」からZIPファイルをダウンロードし、展開してください。

---

### 2. Pythonライブラリをインストールする

必要なライブラリをインストールします。

```bash
pip install opencv-python mediapipe
```

使用環境によっては追加のライブラリが必要になる場合があります。

今後、必要なライブラリを整理した `requirements.txt` を追加する予定です。

---

### 3. Unityプロジェクトを開く

Unity Hubからリポジトリ内の

```text
Unity
```

フォルダを開いてください。

シーンを開き、Playを実行します。

---

### 4. Python側を実行する

Unityを再生した状態でターミナルを開きます。

Pythonフォルダへ移動します。

```bash
cd Python
```

次のコマンドを実行します。

```bash
python main.py
```

Webカメラが起動し、MediaPipe Handsによる手の認識が開始されます。

取得した操作情報はUDP通信によってUnityへ送信され、3Dオブジェクトの拡大縮小や回転に反映されます。

---

### 5. カメラが起動しない場合

使用する環境によってWebカメラ番号が異なる場合があります。

`Python/main.py` 内の

```python
cv2.VideoCapture(1)
```

を

```python
cv2.VideoCapture(0)
```

へ変更すると動作する場合があります。

使用する環境に応じて番号を変更してください。

## 操作方法

### 拡大縮小

両手の距離の変化を利用して、3Dオブジェクトの大きさを変更します。

- 両手を広げる：拡大
- 両手を近づける：縮小

### 回転

手の動きや位置関係を利用して3Dオブジェクトを回転させます。

現在は、操作時のブレや意図しない回転を減らすため、判定方法や回転量の調整を進めています。

今後は、複数の3D編集操作を段階的に扱うため、操作体系に3層構造を導入する予定です。

具体的な各層の役割や操作方法については現在設計・実装を進めています。
